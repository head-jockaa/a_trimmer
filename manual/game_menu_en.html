Anime Trimmer X

- Menu during the game -

We'll explain about the menu when you press C key during the game.

DBS

You catch DBS channels by using a satellite antenna.
when the waves on the sea gets high, anime is televised on DBS.
Moreover, when the waves are glaring, more than 2 anime are televised on DBS.
You choose a channel with arrow key.

Antenna
You can choose the kinds of antenna.
About the antenna, please refer to this page.

Crops now

You can see the list of anime programs you've gathered.
The number of failing to get is shown as well, so let's make sure once in a while.

Save
You save the data of an unfinished game.
You can save a new data without overwriting by choosing "NEW".

Config
You can also modify the configuration during the game.

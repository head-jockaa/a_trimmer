Anime Trimmer X

- Configuration -

Each configuration is changed by Z key or left-right key.
Some items will be added by purchasing at the shop.

Key Setting
You push the keys which you want to set one by one, by turns.
When you made a wrong setting, you can restore it by F4 key.

Radio Tower
This configures whether or not to display radio towers on the map.

Drawing Radio
This configures about the light appeared when anime is televised.
You can choose 4 ways of drawing or "Not draw".

Language
This switches from Japanese to English.
You can switch it by A key at any time,
and what you set here is first shown when you start up this software.

Magnify Map
This makes the map display large during the game.
You can choose from 1 time to 4 times at the beginning.
The speed of movement and the time get slower if you magnify the map.

Accel.
You can choose the way of accelerating the player.
  • Type Mario
    X key increases your speed while you are pressing arrow key.
    you can gather more speed by pressing X key many times.
  • Type Sonic
    You'll speed up gradually if you continue to move with arrow key.
  • Type Kirby
    Beating arrow key will increase your speed.
  • Type Megaman
    X key will cause you to do a sliding quickly while you are pressing arrow key.

    Adjust Dir.
    The antenna will turn to the direction of radio tower automatically
    when you hoist it during the game.
    This function does not work if you set to "Manual".

    Guide
    In this software, an explanation of key-control is displayed at all times on the screen,
    but you can set it to "Not display".

    BGM volume
    This adjusts the volume of music.

    SE volume
    This adjusts the volume of sound effect.

    NHK removal
    This removes the radio towers which have only NHK channels.
    They not only won't be displayed but won't be received.

    Walking
    "Smooth" is the ordinary mode.
    If you choose "Bumpy", A reaction arises while you climb a mountain.

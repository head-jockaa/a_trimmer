Anime Trimmer X

- Various antennas -

Several kinds of antennas are prepared for using in this game.
You can choose Simple Rod and Convenience Rod from the beginning.
Other antennas will be added if you buy them at the shop.
You can choose them by opening Menu during the game.

Simple Rod

This is the most simple antenna that you turn a channel number with arrow key.

Convenience Rod

This shows the table of TV stations receivable above a definite level.
You move the cursor with up-down key and decide it with Z key.
If it's trouble to press Z key, you can use "turn over channels" with left-right key.
I guess this is the most convenient as a game.

UV Rod

This is a classical antenna that separates the dial into VHF and UHF.
It's an antique idea.
You choose the dials with up-down key and turn it with left-right key.

MHz Rod

This is an antenna that adjusts a frequency, in the same way as a radio.
You move a graduation with left-right key.

Button Rod

This is the regular button-style.
You choose a button with arrow key and press it with Z key.
S button will start to set TV channels.
It has "Area code settings", almost good for nothing, as well as "Auto channel settings".
It's just a rummage. Not useful for playing the game.

Super Handy Rod

This is the developed version of Convenience Rod.
This displays the entire radiowaves receivable above a definite level.
This table displays TV station's name, radiowave's power, relay station's name, from the left.
